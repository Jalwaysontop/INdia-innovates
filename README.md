# 🏛️ IntelGraph: Sovereign Strategic Engine
> **A Multi-Agent Framework for Proactive National Governance.**

IntelGraph is a high-performance **Agentic AI Framework** built to handle national-scale strategy. It processes complex global data into actionable insights for **Goals, Policies, and Crises**.

---

## 🚀 The Architecture
IntelGraph operates on a **7-Agent Cabinet System** to ensure high reliability and "Red-Teamed" accuracy.

### 🧠 The Intelligence Layer
| Agent | Role | Technical Focus |
| :--- | :--- | :--- |
| **Sovereign Supervisor** | Orchestrator | Task delegation and final consensus. |
| **Artha Agent** | Economy | Fiscal analysis and trade impact. |
| **Raksha Agent** | Security | Tactical defense and internal stability. |
| **Shakti Agent** | Resources | Energy and supply chain security. |
| **Vikriti Agent** | Red Team | **Adversarial Analysis** and risk mitigation. |

---

## 🛠️ Current Tech Stack
* **Backend:** FastAPI (Python 3.11+)
* **Orchestration:** LangGraph (Stateful Multi-Agent Workflows)
* **Inference:** Groq Llama-3 (Ultra-low latency API)
* **Interface:** Streamlit (Internal Admin & Testing Dashboard)
* **Deployment:** Docker on Hugging Face Spaces
